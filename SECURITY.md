# Security Policy

## Supported versions

This project is pre-1.0 and deployed continuously from `main`. Only the latest
deployed version is supported; fixes land on `main`.

## Reporting a vulnerability

**Please do not report security issues in public GitHub issues.**

Instead, report privately using GitHub's
[**Report a vulnerability**](https://github.com/JoshDoesIT/tennessee-hiking-club/security/advisories/new)
feature (Security → Advisories), which opens a private advisory.

Please include:

- A description of the issue and its potential impact
- Steps to reproduce (proof-of-concept if possible)
- Affected URL/route or file, and any relevant logs

We aim to acknowledge reports within **5 business days** and to provide a
remediation timeline after triage. We'll credit reporters who wish to be named
once a fix is released.

## Scope

In scope: this repository's source, its deployed site, and its CI/CD config.
Out of scope: third-party services we integrate with (Vercel, Stripe, Printful,
OpenStreetMap); report those to the respective vendor. Denial-of-service,
social engineering, and automated scanner noise are not eligible.

Thanks for helping keep the Tennessee Hiking Club community safe.
